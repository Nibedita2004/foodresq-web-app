# 🍽️ FoodResQ – Food Rescue System

FoodResQ is a web-based application that connects food donors with people in need to reduce food wastage. It allows users to donate surplus food, request food, and manage pickups efficiently.

---

## 🚀 Features

* Donate excess food easily
* Request food for NGOs and individuals
* Pickup and distribution management
* User authentication (Login & Register)
* Chatbot support

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express
* **Database:** SQL

---

## 📂 Project Structure

```id="t1n9a0"
public/        → Frontend (HTML, CSS, JS, images)

```

## ▶️ How to Run Locally

1. Clone the repository

```id="x7u3pm"
git clone https://github.com/Nibedita2004/foodresq-web-app.git
```

2. Install dependencies

```id="u9b2k1"
npm install
```

3. Start the server

```id="g5r8n4"
node src/server.js
```

4. Open in browser

```id="c3y6d2"
http://localhost:3000
```
